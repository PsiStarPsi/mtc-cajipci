Address					Read/Write
---------------------------------------------------
0x0:SPI send register			R/W
0x4:SPI recv register			R
0x8:SPI control:
bit	0: spi start 			R/W
	1: spi done			R
	2-3: CDCE select		R/W
	4: CDCE sync			R/W
0x12:ACK and TRG
bit	0-11:  ACK lines		R/W
	16-31: TRG lines		R
0x16:JTAG select
bit	0-3: JTAG select		R/W
